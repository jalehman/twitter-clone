## Twitter Redux

This is a basic twitter app to read and compose tweets the [Twitter API](https://apps.twitter.com/).

Time spent: `9 hours`

### Installation

1. `git submodule update --init`
2. `cd ReactiveCocoa; script/bootstrap; cd ..`
3. `pod install`
